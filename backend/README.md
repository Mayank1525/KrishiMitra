# Backend (Node.js / Express)

This is the main server that connects frontend, ML models, and chatbot.

## Responsibilities
- Create REST APIs
- Connect ML services
- Handle database operations
- Manage authentication (optional)

## APIs
- POST /predict-crop
- POST /predict-price
- POST /chat
- GET /weather

## Tech Stack
- Node.js
- Express.js
- MongoDB

## How to Run
```bash
npm install
node server.js
