# Frontend (React)

This folder contains the user interface for KrishiMitra AI.

## Responsibilities
- Build UI for:
  - Crop Recommendation
  - Price Prediction
  - Dashboard
  - Chatbot Interface
- Integrate APIs from backend
- Implement language selector (Hindi/English)

## Tech Stack
- React.js
- CSS / Tailwind (optional)
- Axios / Fetch API

## How to Run
```bash
npm install
npm start
